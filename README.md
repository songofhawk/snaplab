# SnapLab

A powerful local image editor with AI-powered segmentation, background removal, cropping, resizing, and smart image splitting. No API key required, all processing runs locally in your browser.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
